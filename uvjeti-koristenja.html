Uvjeti Korištenja

Dobrodošli na web stranicu Angelyze ('web stranica'). Molimo vas da pažljivo pročitate sljedeće uvjete korištenja prije nego što pristupite ili koristite našu web stranicu. Korištenjem ove web stranice prihvaćate i slažete se s uvjetima korištenja opisanim u nastavku.

1. Prihvaćanje Uvjeta Korištenja

Korištenjem ove web stranice potvrđujete da ste pročitali, razumjeli i prihvatili ove uvjete korištenja. Ako se ne slažete s bilo kojim dijelom ovih uvjeta, nemojte koristiti našu web stranicu.

2. Intelektualno Vlasništvo

Sadržaj ove web stranice, uključujući tekst, grafike, logotipe, fotografije, audio i video materijale, autorska prava i zaštitni znakovi su vlasništvo Angelyze ili trećih strana i zaštićeni su zakonima o autorskim pravima. Ne smijete kopirati, reproducirati, distribuirati ili koristiti bilo koji dio sadržaja ove web stranice bez prethodnog pismenog odobrenja.

3. Korištenje Web Stranice

Koristite ovu web stranicu isključivo u svrhe koje su zakonite i u skladu s ovim uvjetima korištenja. Ne smijete koristiti ovu web stranicu na način koji može uzrokovati štetu, ometanje ili degradaciju usluga ili narušiti sigurnost web stranice.

4. Privatnost i Sigurnost

Vaša privatnost je važna za nas. Molimo vas da pročitate našu Politiku privatnosti kako biste bolje razumjeli kako prikupljamo, koristimo i štitimo vaše osobne podatke prilikom korištenja ove web stranice.

5. Linkovi na Treće Strane

Ova web stranica može sadržavati linkove na treće strane web stranice. Ne snosimo odgovornost za sadržaj ili politike privatnosti tih web stranica. Preporučujemo da se upoznate s uvjetima korištenja i politikama privatnosti tih web stranica prije nego što ih koristite.

6. Promjene Uvjeta Korištenja

Zadržavamo pravo izmijeniti ove uvjete korištenja u bilo kojem trenutku bez prethodne obavijesti. Svaka promjena bit će objavljena na ovoj web stranici. Vaša nastavljena upotreba web stranice nakon takvih promjena smatrat će se vašim prihvaćanjem novih uvjeta.
